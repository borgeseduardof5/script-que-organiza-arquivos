# File Organizer

Script em Python que organiza automaticamente arquivos de uma pasta
em categorias baseadas na extensão.

## Features

- Organiza arquivos automaticamente
- Cria pastas por categoria
- Modo simulação (--simular)
- Logs no terminal
- Contador de arquivos organizados

## Tecnologias

Python
os
shutil

## Como usar

python organizer.py
